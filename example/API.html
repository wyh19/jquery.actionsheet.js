参数说明
参数名 类型 默认值 说明 版本
mode string list 菜单格式，有list和grid两种参数，如果是grid，必须有图片字段 1.0.0
url string 空字符 远程数据地址，返回结果可以是json字符串或json对象 1.0.0
data array null 本地数据，数组 1.0.0
imgBaseUrl string 空字符 图片根路径 1.0.0
imgField string 空字符 图片地址字段 1.0.0
labelField string 空字符 菜单文字字段 1.0.0
idField string 空字符 菜单id字段 1.0.0
clickType string click 点击事件类型，默认是click，也支持tap或其他自定义事件名 1.0.0
debug bool false 开发时可以使用此参数看到错误信息 1.0.1
backgroundColor string #333 背景色 1.0.1
color string #fff 文字颜色 1.0.1
triggerFirst bool false 当mode设置为grid时，该参数有效，当为true时，自动点击第一个菜单项 1.0.1
callback function(id,label) null 点击菜单的回掉事件，用于自定义点击菜单的效果 1.0.0

方法说明
方法名 参数 返回值 说明 版本
show 无 无 显示菜单栏 1.0.0
hide 无 无 隐藏菜单栏 1.0.0
reRender 无 无 重新绘制菜单 1.0.2
getVersion 无 版本号 返回版本号，主要为应对API文档 1.0.2
